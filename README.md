# Firebase + VueJS Blog

Serverless blogging app 

| Backend | Firebase |
| Frontend | VueJS |
| CSS | Bulma |

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```
